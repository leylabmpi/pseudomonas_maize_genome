============
Installation
============


Stable release
--------------

To install pseudomonas_maize_genome, run this command in your terminal:

.. code-block:: console

    $ pip install pseudomonas_maize_genome

This is the preferred method to install pseudomonas_maize_genome, as it will always install the most recent stable release. 

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for pseudomonas_maize_genome can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/nick-youngblut/pseudomonas_maize_genome

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/nick-youngblut/pseudomonas_maize_genome/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/nick-youngblut/pseudomonas_maize_genome
.. _tarball: https://github.com/nick-youngblut/pseudomonas_maize_genome/tarball/master
